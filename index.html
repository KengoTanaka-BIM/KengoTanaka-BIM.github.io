自己紹介

スーパーゼネコン案件などでRevitアドイン開発や多国籍設備モデリングチーム管理を経験。 ISO 19650に基づくデータ整理やモデリング業務を通じて、効率的な設計支援を行ってきました。 現在はMEPモデリング管理とRevitアドイン開発に注力し、業務改善と生産性向上を目指しています。

アドイン実績

ChangeLens

Revitモデルの追加・変更・削除要素を色分け表示し、差分をExcelに書き出すRevitアドイン。作業進捗管理やISO19650ログ管理に活用。



GitHub Qiita



PipeDuctQuantifier

Revitモデル内の配管・ダクトの数量を自動で集計・エクセルデータでエクスポートするRevitアドイン。

　　　　　　

GitHub Qiita



ElementParameterExporter

Revitモデル内で選択した要素のパラメータ情報をCSVでエクスポートするRevitアドイン。

GitHub Qiita



ElementParameterImporter

要素パラメータをCSVでインポートするRevitアドイン。

GitHub Qiita



ViewSheetPairing

指定したビューを指定したシートへ一括配置するRevitアドイン。
